Question 1

Create a graph that shows the distribution of companies in the dataset by State (ie how many are in each state). There are a lot of States, so consider which axis you should use. This visualization is ultimately going to be consumed on a portrait oriented screen (ie taller than wide), which should further guide your layout choices.

inc %>% 
count(State) %>% 
ggplot(aes(x = reorder(State,n,sum),y= n , fill = n))+  geom_col()+ labs(title = "Count of Companies By State", x = "State", y = "Count Of Companies") + coord_flip()

Quesiton 2

Lets dig in on the state with the 3rd most companies in the data set. Imagine you work for the state and are interested in how many people are employed by companies in different industries. Create a plot that shows the average and/or median employment by industry for companies in this state (only use cases with full data, use Rs complete.cases() function.) In addition to this, your graph should show how variable the ranges are, and you should deal with outliers.

df <- inc %>%
filter(complete.cases(.)) %>%
group_by(State) %>%
mutate(companies = n()) %>% 
arrange(desc(companies))%>%
ungroup() %>%
mutate(y = dense_rank(desc(companies))) %>%
filter(y == 3) %>%
group_by(Industry)

Question 3

Now imagine you work for an investor and want to see which industries generate the most revenue per employee. Create a chart that makes this information clear. Once again, the distribution per industry should be shown.

df3 <- subset(inc, complete.cases(inc)) %>%
mutate(median_emp = (Revenue/Employees)/1000)
  

ggplot(df3, aes(factor(x = reorder(Industry, median_emp, FUN = median)))) +
stat_summary_bin(aes(y = median_emp), fun.y = "median", geom = "bar",fill ="blue") +
coord_flip() + xlab("Industry") + ylab("Average Revenue Per Employee") + theme(axis.text.x = element_text(hjust = 1))
